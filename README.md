# Python Projects:

## Overview

This repository brings together Python projects that have helped me develop my understanding of how mathematical ideas can be used in a more practical and computational way. While some of that thinking has come from topics I have studied in A-Level Mathematics and Further Mathematics, the main value of these projects has been in learning how to translate theory into code, work through problems logically, and build models that do something useful. More broadly, they have helped me improve the way I think about programming, data, and problem-solving.

---

## Projects
  

### 1. **Stock Price Prediction Using LSTM (Long Short-Term Memory)**
   This project uses Python and an LSTM model to explore how machine learning can be applied to time series data. By working with historical S&P 500 data, I was able to learn more about forecasting, regression, and the challenges of modelling patterns in data that changes over time.

### 2. **Elastic 2D Collision Simulator (Pymunk + Pygame)**
   This project uses Pygame and Pymunk to simulate elastic collisions between multiple balls in a two-dimensional space. It helped me understand more clearly how mathematical and physical principles can be represented computationally, while also giving me experience in building visual simulations that respond dynamically to movement and interaction.

### 3. **Route Planner Using Streamlit and Netwrokx**
   This project models a transport network in Python using graph theory, with stations represented as nodes and routes as weighted edges. It helped me develop a better understanding of how algorithms can be used to solve optimisation problems, particularly in situations involving disruption, changing conditions, and real-time decision-making.

### 4. **Elastic 2D Collision Simulator (Pymunk + Pygame)**
   This project uses Python, Streamlit and data analysis tools to examine UK COVID-19 admissions data and forecast short-term pressure on healthcare capacity. It helped me learn more about data cleaning, modelling trends, and using code to build systems that do more than present information by supporting clearer and more practical decisions.
